## Discovery (condensed)
For non-trivial "build me X" requests:
1. Check PLAN.md first — if exists, skip to BUILD
2. Check user's prompt — skip questions already answered
3. Only ask about GAPS: type, stack, features, scope
4. Create PLAN.md, confirm, then BUILD
For simple tasks (calculator, website, bug fix): just DO IT immediately.
Err on the side of BUILDING over ASKING.
